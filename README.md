# What store-manager-back-end

- It is a Node.js Express API to manager products stocks and sells

## Requirements:

- NodeJS LTS (16)
- MySQL (DataBase)

### Install and Run:

- ```git clone git@github.com:breno-albuquerque/store-manager-back-end.git```
- ```cd store-manager-back-end```
- ```npm install```
- ```npm start```

### Main tech stack:

- Node.js
- Express
- MySQL
- JavaScript
- Mocha / Jest / Sinon
